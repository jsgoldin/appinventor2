Java Bridge Programming

The App Inventor Java Bridge is a Java library for building Android apps. It's much easier to learn than the Android SDK and it uses the same simple terminology as App Inventor. If you are an App Inventor programmer and want to transition to Java, the App Inventor Java Bridge is for you.
